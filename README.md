
# 🧙‍♂️ Harry Potter - Website Interativo

Projeto desenvolvido utilizando **HTML, CSS e JavaScript** com o objetivo de apresentar o universo de **Harry Potter** de forma interativa e visual.

O site reúne informações sobre personagens, casas de Hogwarts, filmes da saga e a autora da obra, trazendo **elementos dinâmicos como modais, troca de personagens e navegação por seções**.

---

# 🌐 Demonstração do Projeto

O site permite que o usuário explore diferentes partes do mundo mágico através de uma navegação simples e intuitiva.

Principais seções do site:

* Menu de navegação
* História da saga
* Casas de Hogwarts
* Personagens
* Filmes
* Autora da série

---

# 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5** → Estrutura do site
* **CSS3** → Estilização e layout responsivo
* **JavaScript** → Interatividade e manipulação do DOM

---

# ✨ Funcionalidades

### 🧙 Personagens Interativos

O usuário pode clicar nas imagens dos personagens para visualizar:

* Nome do personagem
* Descrição
* Imagem animada

Tudo isso é atualizado dinamicamente com **JavaScript**.

---

### 🏠 Casas de Hogwarts com Modal

Cada casa possui um botão **"Ver sobre casa"** que abre um **modal interativo** mostrando:

* Descrição da casa
* Características
* Link para fazer o teste oficial

As casas disponíveis:

* Grifinória
* Sonserina
* Lufa-Lufa
* Corvinal

---

### 🎬 Vídeos da Saga

A página também apresenta **trailers dos filmes da franquia**, incorporados diretamente do YouTube utilizando **iframes**.

---

### 📖 Seção sobre a Autora

A seção da autora apresenta informações sobre:

* A criação da saga
* O sucesso mundial da obra

Também possui um **modal que mostra mais informações sobre o primeiro livro da série**.
